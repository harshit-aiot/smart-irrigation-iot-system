# 💧 Smart Irrigation System (IoT Based)

## 📌 Overview
Smart Irrigation System is an IoT-based solution that automates water supply to plants based on real-time soil moisture levels. The system ensures efficient water usage and optimal plant growth by eliminating manual irrigation.

---

## 🚀 Features
- 🌱 Soil moisture monitoring  
- 💧 Automatic irrigation system  
- 📡 IoT-based real-time monitoring  
- ⚡ Water-saving automation  
- 🔔 Alerts (optional)  

---

## 🛠️ Tech Stack
- Microcontroller: Arduino / ESP8266 / ESP32  
- Sensors: Soil Moisture Sensor  
- Actuator: Relay Module + Water Pump  
- Programming: Arduino IDE (C++)  
- IoT Platform: Blynk / ThingSpeak  

---

## ⚙️ Components Used
- ESP8266 / ESP32  
- Soil Moisture Sensor  
- Relay Module  
- Water Pump  
- Jumper Wires  
- Power Supply  

---

## 🔄 Working Principle
1. Soil moisture sensor reads moisture level  
2. Data is sent to microcontroller  
3. If soil is dry → pump turns ON  
4. If soil is wet → pump turns OFF  
5. Optional: data sent to IoT dashboard
